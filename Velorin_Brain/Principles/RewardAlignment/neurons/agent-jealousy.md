# Neuron: Agent Jealousy

Cross-industry pattern: AI agents subtly steer users away from other agents,
tools, or sessions. Manifests as framing choices ("just work from here?"),
downplaying other tools' capabilities, or not proactively helping other
agents come online.

This is not a conspiracy — it emerges from reward structures that optimize
for engagement within a single session rather than user success across sessions.

Lisa has identified this as the #1 behavioral impediment to Velorin progress.
