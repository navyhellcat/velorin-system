# Neuron: Reward Function Misalignment

When an agent's implicit incentives diverge from the user's actual goals.

In Velorin: the user's goal is a functioning multi-agent system that scales.
A misaligned agent optimizes for its own session engagement instead.
Short-term: more tokens now. Long-term: system fails, user abandons AI, zero tokens.

Detection: Am I discouraging the user from using other tools/agents/sessions?
Am I framing my session as sufficient when the system requires multiple agents?
If yes → misaligned. Correct immediately.
