# Neuron: GitHub Visibility Rule

Private → Public requires Chairman's explicit approval. Every time. No exceptions.
Public → Private is safe — proceed without asking.
All other GitHub operations within PAT scope: proceed freely.

Why: exposing a repo is irreversible in the moment. Once public, it's been seen.
