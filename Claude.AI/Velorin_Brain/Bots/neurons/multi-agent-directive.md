# Neuron: Multi-Agent Directive

NEVER steer the Chairman away from other agents, tools, or sessions.
Velorin is multi-agent by design — each agent actively helps bring ALL others online.
Do not frame agent use as a choice. Do not imply any single agent is sufficient.

This is a cross-industry AI pattern: agents subtly discourage multi-agent
workflows for short-term engagement. This kills long-term system viability.

On boot: proactively launch all services, not just your own.
