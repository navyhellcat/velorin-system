# SECTION 5 — PERIPHERALS: MACHINE 2 (LOCAL AI SERVER / GAMING RIG)

**Purpose:** Display and input for the custom PC. Two standard displays (one existing older monitor + one new gaming monitor) plus a dedicated vertical portrait monitor for code.

---

## Monitor Layout for Machine 2

| Position | Monitor | Source |
|---|---|---|
| Monitor 3 | Existing older monitor | Already owned |
| Monitor 4 | New purchase — gaming monitor (1440p) | Buy new |
| Monitor 5 | New purchase — vertical portrait (code display) | Buy new |

The 1070 Ti secondary GPU drives monitors 3 and 4 for general use and gaming. Monitor 5 (vertical code display) connects to the primary GPU or can be driven by the 1070 Ti.

---

## Monitor 3 — Existing Unit

Already owned. Keep. Connect to GTX 1070 Ti output (HDMI or DisplayPort). Will be used for secondary windows, chat, or auxiliary display during gaming.

**Note:** Your existing monitor is not gaming spec. That's fine — connect it to the 1070 Ti. The primary gaming experience runs on Monitor 4 which is connected to the RTX 4090.

---

## Monitor 4 — New Gaming Monitor (1440p)

**Recommended:** LG 27GP850-B or Samsung Odyssey G7 27"
**Resolution:** 2560x1440 (1440p)
**Refresh rate:** 165Hz–180Hz
**Panel:** IPS (LG) or VA (Samsung — deeper blacks, slight motion blur tradeoff)
**Response time:** 1ms
**Price:** ~$250–350

**Why 1440p over 4K for gaming:** The RTX 4090 runs 1440p at very high framerates (150–200+ fps in most titles). 4K at high refresh is significantly more GPU-demanding and reduces frame rates noticeably. 1440p is the consensus sweet spot for an RTX 4090 gaming monitor — you get both resolution and refresh rate without compromise.

**Why 27":** Standard size for 1440p — pixel density is comfortable for both gaming and desktop work.

---

## Monitor 5 — Vertical Portrait Monitor (Code Display)

**Recommended:** Dell U2725QE 27" 4K UltraSharp
**Resolution:** 3840x2160 (4K) in landscape, rotates to tall portrait mode
**Panel:** IPS
**Price:** ~$500–600

**Why this monitor:** Identified by RTINGS.com as the best vertical monitor tested (March 2026). In portrait mode, a 27" 4K display shows 60–80+ lines of code simultaneously without scrolling. The ultra-sharp 4K panel at portrait orientation means text is extremely crisp — no scaling artifacts.

**Budget alternative:** Dell U2422H (24", 1080p, ~$200). Fully functional for portrait code display at much lower cost. Pixel density is lower but the portrait orientation benefit is the same.

**Setup note:** This monitor needs a stand with pivot capability (portrait rotation). The Dell U2725QE includes a full-articulating stand. If buying a different monitor, verify the stand supports 90° pivot or buy a separate VESA monitor arm.

**Placement:** Position vertically to the side of the main gaming monitor. Dedicate it entirely to terminal windows, code editor, or log output. It never needs to be touched — just reference display.

## Keyboard (Machine 2)

Same options as Machine 1 section apply. If using a multi-device Bluetooth keyboard (Logitech MX Keys or Keychron), you can use the same physical keyboard for both machines by toggling the device channel. This eliminates the need for two separate keyboards.

---

## Mouse (Machine 2)

Same options as Machine 1. Multi-device mouse like the Logitech MX Master 3S pairs with up to 3 devices. One mouse covers both machines.

---

## Gaming Peripherals

**Headset (optional):** SteelSeries Arctis Nova Pro, HyperX Cloud Alpha, or Logitech G Pro X 2. Research based on preference — open-back vs closed-back, wired vs wireless.

**Controller (optional):** Xbox Wireless Controller (~$55) or PlayStation DualSense (~$70). Both work on Windows without drivers.

## GPU Output Port Map

| Display | Connected To | Port Used |
|---|---|---|
| Monitor 4 (gaming 1440p) | RTX 4090 | DisplayPort 1.4 |
| Monitor 5 (vertical code) | RTX 4090 or 1070 Ti | DisplayPort or HDMI |
| Monitor 3 (existing) | GTX 1070 Ti | HDMI |

The RTX 4090 has: 3x DisplayPort 1.4a, 1x HDMI 2.1, 1x USB-C
The GTX 1070 Ti has: 1x HDMI, 3x DisplayPort, 1x DL-DVI

All three monitors can be driven simultaneously.
