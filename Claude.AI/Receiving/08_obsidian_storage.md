# ZONE 8 — OBSIDIAN AS VELORIN KNOWLEDGE INFRASTRUCTURE

Thesis: Obsidian vaults are the best substrate for personal AI knowledge bases — local-first, plaintext, graph-structured, user-controlled.

Architecture: Obsidian Vault ↔ MCP Server ↔ Claude Code/Desktop. No boot protocol. No Drive fetch. No token cost on reads.

Migration path: Install Obsidian → mirror Drive structure → install obsidian-skills + MCP server → Claude Code reads natively → Drive becomes backup → CLAUDE.md replaces boot protocol.

Replaces: Drive boot protocol, [VELORIN.EOF] checking, manual uploads, session handoffs, Google Docs knowledge web.
