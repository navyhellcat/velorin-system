## OPEN QUESTIONS THE GOAL MAP CANNOT ANSWER

These require decisions before Phase 3 can proceed:

**OQ1: Can CT's intelligence be encoded?**
- If yes: this goal map executes.
- If no: by Phase 3, pivot to enterprise where partial context still produces useful output for explicit strategic questions.
- Decision point: end of Phase 2. If intake test v0 spec cannot articulate how CT's cross-domain reasoning gets captured, raise the flag before building Phase 3.

**OQ2: Who is the first external user?**
- Cannot be a stranger. Must be someone CT knows well enough to do the intake himself.
- The first run of the intake is a design test as much as a product test. CT is the intake interviewer in Phase 3.
- This changes the sales motion: Phase 3 is not marketing. It's CT reaching into his network.

**OQ3: What is the minimum intake?**
- Layer 3 left this open. If the intake takes 90 minutes and produces one irreplaceable insight, prosumer is viable.
- If the intake takes 3 hours and produces nothing until month 2, prosumer fails. Enterprise required.
- This must be answered by Phase 3.1. Jiang's research must include benchmarks on how fast personal intelligence accumulates.

---

## PHASE GATE LOGIC

If Phase 2 (intake v0 spec) produces a credible answer to OQ1 → proceed to Phase 3.
If Phase 3.7 (first-session cross-domain output) fails → return to 2.6 (intake redesign).
If Phase 3.8 (first paying user) fails after redesign → reassess B2C vs enterprise at CT's next strategic review.
If Phase 4.6 (exit decision) fails (revenue insufficient) → CT remains at Camfil for Q1 2027 while scaling to 10 users.

The goal map has gates. It does not assume linear success. Each gate is a deliberate decision point.
