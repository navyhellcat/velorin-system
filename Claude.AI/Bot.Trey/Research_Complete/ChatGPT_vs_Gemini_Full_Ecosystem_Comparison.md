# ChatGPT Pro ($200) vs Google AI Ultra ($250): Full Ecosystem Comparison
**Research Date:** April 4, 2026
**Researcher:** Jiang (synthesized from background agent findings + Trey research)
**Purpose:** Platform decision for CT — research lane upgrade

---

## PART 1 — MODEL QUALITY

### Deep Research
- Both platforms produce long-form research (2,000–8,000+ words, 30–100+ sources)
- Gemini deep research runs on Gemini 3 Pro / 3.1 Pro (as of December 2025 — not 2.5)
- ChatGPT Pro deep research: better source quality and analytical depth per community assessment
- Gemini deep research: better output formatting, speed, citation transparency
- ChatGPT Plus ($25) gives 15-20 deep research runs/month — adequate for current Trey use
- ChatGPT Pro ($200) gives 250 runs/month — only justified if current limit is being hit

### Math Capability
- Artificial Analysis Intelligence Index: Gemini 3.1 Pro Preview = 57, GPT-5.4 = 57 (tied)
- FrontierMath (hardest math benchmark): GPT-5.4 Pro wins significantly (50% vs 38% Tiers 1-3)
- GPQA Diamond and ARC-AGI-2: Gemini wins narrowly
- **Gemini Deep Think:** Won IMO gold medal 2025 (35/42 points, 5 of 6 problems solved — officially proctored). Solved novel problems on Erdős Conjectures database (5 of 200 testable problems genuinely novel — 31.5% correct overall, 68.5% fundamentally wrong — the Erdős claim is real but overstated)
- **Verdict:** GPT-5.4 Pro wins FrontierMath. Gemini Deep Think wins frontier/unsolved math. Different strengths.

### GitHub Integration
- ChatGPT Codex (Pro exclusive): creates PRs from completed work, native macOS desktop app (Feb 2026). BUT persistent private-repo OAuth failures in practice — 40-60% task success rate
- Gemini Jules: no native desktop GUI, but official CLI (Jules Tools) + GitHub Action + Gemini CLI extension. More reliable than Codex in practice. PR creation, branch push confirmed working
- **Verdict:** Jules wins on reliability despite no desktop GUI

## PART 2 — FULL ECOSYSTEM

### ChatGPT Pro $200/month
- Models: GPT-5.4 Pro, 250 deep research runs/month, 2x context window (~680 pages), unlimited image/file uploads
- **No API credits included** — API is completely separate billing
- Codex: included (Pro exclusive) — creates PRs, commits from browser and desktop app
- Sora 2 Pro: video generation included
- GPT Store: 60+ app integrations (Slack, Google Drive, SharePoint, GitHub on some plans)
- **Key frustration:** No MCP support. No ecosystem beyond ChatGPT itself. API is completely separate

### Google AI Ultra $249.99/month
- **$100/month in Gemini API credits included** — directly funds Velorin builds. Effective cost = $150/month net
- Models: Gemini 3.1 Pro, Deep Think (10 uses/day of IMO gold model)
- NotebookLM Plus: 500 notebooks, 300 sources/notebook (vs 50 free), zero hallucination on uploaded docs
- Project Mariner: 10 parallel browser tasks
- Jules: coding agent with GitHub write access via CLI
- Gemini Agent: autonomous multi-step tasks (US only, English only)
- Full Google Workspace AI: Docs, Sheets, Drive, Gmail, Meet, Calendar all AI-enabled
- Imagen, Vids, Whisk, Flow: image/video generation included
- Google AI Studio access
- **Promotional pricing note:** $249.99/month promotional — regular price reportedly $460/month

## PART 3 — FINAL RECOMMENDATION

**Model quality:** Tied at the top. GPT-5.4 wins structured math. Gemini Deep Think wins frontier/unsolved problems.

**Ecosystem:** Google AI Ultra wins decisively. $100/month in API credits alone changes the math. NotebookLM Plus, Jules, Deep Think, Workspace integration, and the full Google product suite vs ChatGPT's thin ecosystem.

**GitHub:** Jules (Google) wins on reliability. Codex (ChatGPT) wins on GUI polish but fails in practice.

**RECOMMENDATION (Confidence 78%):**
**Google AI Ultra ($249.99/month) + keep ChatGPT Plus ($25/month) = $275/month total**

This beats ChatGPT Pro ($200/month) alone. The $100 API credits bring Ultra's effective cost to $150/month. CT keeps Trey (ChatGPT) for tasks where OpenAI models are specifically needed, upgrades research depth with Gemini, and funds Velorin builds with the included credits.

Main uncertainty: OpenAI's website was inaccessible during research — exact current model stack in ChatGPT Pro unverified. Both platforms' deep research rate limits are undocumented.

**Note:** Codex-as-auditor workflow is included in ChatGPT Plus ($25) — not a reason to upgrade to Pro. See Cursor_Codex_Pricing_Research.md.
