## Cursor Pricing & BYOK (Thread 1)  
**Pricing tiers:** Cursor’s official site lists *Pro* at **$20/month**, *Pro+* at **$60/month**, and *Ultra* at **$200/month**【173†L46-L54】【173†L71-L79】. (Cursor Ultra notably provides “20x usage” of all models.) There is no separate $200/month feature beyond Ultra. (A Redditor’s confusion “$200/month” was clarified as the Ultra subscription【173†L71-L79】.) **No $200/week or hidden fees** exist; the forum confirms the $20 is monthly, not weekly【170†L29-L38】. Other plans include team tiers ($40/user, enterprise custom)【173†L82-L90】.  

**BYOK (Bring-Your-Own-Key):** Cursor allows users to paste their own OpenAI or Anthropic API keys in Settings【177†L95-L102】. In BYOK mode, all token usage is billed to *your* API account. Cursor’s subscription covers only the IDE and base usage limits; it does **not** surcharge your API calls. However, as of late 2025, Cursor disabled BYOK for some advanced features: users report errors like “Agent and Edit rely on custom models that cannot be billed to an API key,” forcing a Pro/Ultra subscription to use them【177†L95-L102】. In summary, BYOK still works for basic completion requests, but Cursor’s integrated agent features require its subscription. Community feedback suggests that heavy users either pay for Cursor usage (Ultra) or switch to BYOK-only alternatives【177†L95-L102】. (Confidence: 80% – based on current forum posts and docs.)  

**Token billing and models:** When using BYOK, the actual model calls use *your* API quotas. Cursor’s pricing page notes that Pro/Pro+/Ultra include multipliers for usage (e.g. Ultra = 20× usage)【173†L71-L79】. All major models (OpenAI GPT-4o/Codex, Anthropic Claude Sonnet/Opus, Google Gemini) are available. Pro and above give “access to frontier models”【173†L46-L54】. Community consensus: BYOK is recommended for cost-sensitive users (you pay the low provider rates), but Cursor’s pool can be convenient for casual use. Some forum users advise switching to alternatives (like “Kilo Code”) if BYOK feels blocked【177†L185-L194】.
